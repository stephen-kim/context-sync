# 페르소나

페르소나는 Claustrum이 각 사용자의 컨텍스트에 가중치를 부여하는 방식을 제어합니다.
이는 권한이나 역할 멤버십이 아닌 검색 강조를 변경합니다.

## 페르소나 모드

- `neutral`(기본값): 약간의 활성 작업 향상으로 균형 잡힌 검색입니다.
- `author`: 구현 중심; 활동적인 작업과 최근 활동을 선호합니다.
- `reviewer`: 위험과 정확성에 중점을 둡니다. 제약과 결정을 선호합니다.
- `architect`: 시스템 설계 중심; 결정과 제약을 선호합니다.

## 권장사항(힌트만)

끝점:

- `GET /v1/context/persona-recommendation?workspace_key=...&project_key=...&q=...`

행동:

- 규칙 기반 추천.
- 사용자 페르소나를 자동으로 전환하지 않습니다.
- 사용자 페르소나는 수동으로 변경될 때까지 정보 소스로 유지됩니다.

## 흐름 적용

1. **컨텍스트 디버그**를 엽니다.
2. 현재 쿼리로 추천을 실행합니다.
3. 원하는 경우 **권장 페르소나 적용**을 클릭합니다.

적용된 변경사항은 `user.persona.changed`으로 감사됩니다.
