# 컨텍스트 디버그 가시성

Claustrum에는 관리 UI에 관리 중심 컨텍스트 디버그 표면이 포함되어 있습니다.

## 관리자 UI 화면

프로젝트 -> 컨텍스트 디버그

특징:

- 번들 미리보기(`default` / `debug`)
- 해결된 작업공간/프로젝트 및 모노레포 모드
- 현재 하위 경로 신호
- 디버그 모드에서 결과별 점수 분석
- 최근 의사결정 추출 결과(결과, 신뢰도, 오류)

## 목적

- 메모리/결과가 선택된 이유를 설명하세요.
- 모노레포 모드 및 하위 경로 처리를 검증합니다.
- 추출기 품질 및 순위 드리프트를 진단합니다.

## 메모

- 디버그 보기는 관찰 및 조정을 위한 것입니다.
- 원본 전체 텍스트 콘텐츠는 계속 보호됩니다. 번들은 간결한 스니펫과 증거 참조를 보여줍니다.
